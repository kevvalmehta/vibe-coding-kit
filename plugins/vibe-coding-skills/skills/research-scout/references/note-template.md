# Research note template → save as `research/<topic>.md`

Copy this shape. Every factual claim MUST carry a real source link. No source = cut it or mark it
"unverified". Never invent a source.

```markdown
# Research: <the question, in plain words>

**Date:** <YYYY-MM-DD>  ·  **Depth:** quick | standard | deep  ·  **Searches used:** <n> / <ceiling>

## Summary — short answer (plain English)
<2–4 sentences: the common approach + the recommendation, for a non-technical reader.>

## Findings
Each finding = one claim + its source(s) + how trustworthy it is.

- **<claim>** — <source title> (<tier: primary | reputable | anecdote>, confidence: high/med/low)
  <url>
- **<claim>** — ...

## Where sources disagree
<List any conflicts and both sides. If none, write "No major disagreement found.">

## Trade-offs
<The honest pros/cons of the main options found.>

## Recommendation
<What this suggests for the owner's decision — and how strongly (given the evidence quality).>

## Sources
1. <title> — <url> (<tier>)
2. ...

## Gaps / what wasn't found
<Anything you couldn't verify, or couldn't reach. Be honest — do not paper over it.>
```

Rules baked into the shape: cite every claim, label source tier + confidence, surface disagreement,
and be honest about gaps (Truth over confidence — Principle VII).
